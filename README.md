wdd130 - Web Fundamentals 🚀

This repository contains the assignments and projects developed as part of the WDD130: Web Fundamentals course at BYU Pathway University. This course was a hands-on introduction to the world of web design and development, exploring the basics of website creation and the diverse careers in the field.

About the "Web Fundamentals" Course 📚
The WDD130 course focused on:

- Introduction to the World Wide Web: Understanding how the web works.
- Basic Web Design and Development: Creating simple, functional web pages.  
- Career Exploration: Gaining insight into different roles and paths in web design and development.
  
It was a practical experience where I learned to build from the ground up, laying the foundation for future studies in the area.
